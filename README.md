# SIDOX — Formation AutoCAD & Covadis Topographie

## 🎯 Objectif du projet
Site web professionnel multi-pages pour la vente de formations en ligne dans les domaines **AutoCAD 2D** (orienté Topographie) et **Covadis**. Conçu pour convertir les visiteurs débutants en inscrits via un design premium et une hiérarchie visuelle claire.

## 🎨 Système de design

### Palette de couleurs
- `--bg: #070e1a` — Fond principal
- `--accent: #e8213a` — Rouge SIDOX (CTA, prix)
- `--blue: #4d9cff` — Bleu technique (Covadis, highlights)
- `--gold: #f5c94e` — Or (Offre duo)
- `--text-muted: #8ba3c8` — Texte secondaire

### Typographies
- **Space Grotesk** — Titres principaux, montants, logos
- **Inter** — Corps de texte, navigation, boutons

### Composants
- `.btn`, `.btn-primary`, `.btn-outline`, `.btn-ghost`, `.btn-gold`
- `.card`, `.module-card`, `.program-tile`, `.offer-card`
- `.badge`, `.eyebrow`, `.check-list`
